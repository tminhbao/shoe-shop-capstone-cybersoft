- Đăng nhập
- Đăng nhập bằng Facebook
- Đăng ký
- Thay đổi thông tin người dùng
- Thay đổi mật khẩu
- Thêm sản phẩm vào giỏ hàng
- Submit Order
- Yêu thích sản phẩm
- Thay đổi số lượng sản phẩm trong giỏ hàng
- Submit Order
- Xem Order History
- Xem Favorite List
